# Welcome to Basecamp

Basecamp is a modern component library designed to help you build beautiful, accessible, and performant applications. Built with React, TypeScript, and Tailwind CSS, it provides a comprehensive set of components and primitives that follow modern design patterns.

## What's Included

Basecamp includes:

- **Components**: Ready-to-use UI components with multiple variants and examples
- **Primitives**: Basic building blocks for creating more complex components
- **TypeScript**: Full TypeScript support with comprehensive type definitions
- **Accessibility**: Built with accessibility in mind, following WCAG guidelines
- **Customizable**: Easy to customize and extend to match your design system

## Getting Started

To get started with Basecamp, you can:

1. Browse the [components](/docs/components) to see what's available
2. Explore the [primitives](/docs/primitives) for basic building blocks
3. Check out the examples to see how each component works

## Design Philosophy

Basecamp follows these core principles:

- **Composable**: Components are built to work together seamlessly
- **Accessible**: Every component is built with accessibility in mind
- **Customizable**: Easy to customize and extend
- **Performant**: Optimized for performance and bundle size
- **Modern**: Built with modern React patterns and TypeScript
