# Button

A versatile button component that supports multiple variants and sizes. Built with accessibility in mind and follows modern design patterns.

## Usage

The Button component is the foundation for all interactive elements in your application.

```tsx
import { Button } from "@/components/ui/button/button"

;<Button>Click me</Button>
```
