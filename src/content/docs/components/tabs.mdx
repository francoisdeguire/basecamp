title: Tabs
description: A set of layered sections of content—known as tab panels—that are displayed one at a time.
---

A Tabs component built with React Aria Components for accessible tab navigation.

## Usage

<ComponentPreview component="tabs" example="basic" />

## Features

- **Accessible**: Built with React Aria Components for full accessibility support
- **Keyboard Navigation**: Navigate tabs with arrow keys, Home, End, and Space/Enter
- **Focus Management**: Automatic focus management when switching tabs
- **Flexible**: Support for both horizontal and vertical orientations
- **Customizable**: Extensible variant system for styling
